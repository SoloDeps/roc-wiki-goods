Privacy Policy

This Chrome extension, built with wxt, respects your privacy. Below is detailed information on how it handles data.

Data Collection

This extension does not collect, transmit, or share any personal data. The only data stored locally on your device is:

  • Dropdown selections related to predefined building names from the game Rise of Cultures.

These values are stored using localStorage and are only used for the proper functioning of the extension.

Data Usage

The stored data is used exclusively to:

  • Keep user selections saved between sessions.
  • Update game-related icons dynamically based on user choices.

Third-Party Sharing

This extension does not share or transmit any data to third parties, servers, or external services.

Data Deletion

You can clear the stored data at any time by manually resetting the dropdown selections or clearing your browser’s local storage.

Contact

If you have any questions or concerns about this privacy policy, feel free to contact us via the extension’s support page or Discord.
